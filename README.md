# Portfolio

Open `index.html` in a browser to view the portfolio.

## Edit Content

Most changes are in `data.js`:

- Change name, role, email, phone, and intro at the top.
- Add or remove skills in the `skills` list.
- Add education items in the `education` list.
- Add experience items in the `experience` list.
- Add or remove projects in the `projects` list.
- Add blog posts in the `blog` list.
- Update social links in the `socials` list.

## Features

- Light / dark mode button.
- Full one-page layout: Home, About, Education, Experience, Skills, Projects, Blog, Contact.

## Change Photo

Replace `assets/profile.jpg` with a new image using the same file name.

## Change Style

Edit `styles.css`.

Useful color variables are at the top:

- `--primary`
- `--primary-dark`
- `--accent`
- `--bg`
- `--ink`
